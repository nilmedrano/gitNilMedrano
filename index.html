Proves Git

  • Pas 1: Inicialitzar el repositori: git init
  • Pas 2: Afegir tots els fitxer: git add .
  • Pas 3: Pujar els canvis al repositiori: git commit -m "Pujar canvis"
  • Pas 4: Crear una branca: git branch nova_branca
  • Pas 5: Canviar la branca utilitzada: git checkout nova_branca
  • Pas 6: Utilitzar un repositori remot a GitHub: git remote add origin https://github.com/nomUsuari/nomRepositori.git
